# Woowacourse Tiny CRA

- webpack
- react
- styled-component
- eslint
- prettier
- typescript (optional)

## Usage

```sh
// install
yarn add woowacourse-tiny-cra
```

```sh
// javascript
node node_modules/woowacourse-tiny-cra/src/index.js

// typescript
node node_modules/woowacourse-tiny-cra/src/index.js typescript
```
